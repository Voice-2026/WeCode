# Changelog

## [0.1.0] - 2026-04-28

### Added

- Initial Codux relay service release for macOS host and mobile client pairing.
- Added host registration, one-time QR pairing, pairing confirmation/rejection, device revocation, and host/client WebSocket forwarding.
- Added end-to-end encrypted secure message relay metadata support for Codux macOS and Codux Mobile.
- Added TOML configuration support for server address, database path, pairing TTL, shutdown timeout, and HTTP read-header timeout.
- Added GitHub Actions release builds, checksum artifacts, and Docker image publishing.
